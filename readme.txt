MultiPaste

Script for Adobe Indesign CS4+

Pastes the contents of the clipboard to each of these pages in the current document. Click menu Edit->MultiPaste. Select from the list page, and then click OK. Hold down CTRL (or Command on Mac) to select multiple pages. Click again to deselect the list item.

Hold the keys before running the script for additional functions:

Shift - selects all the pages
Ctrl - selects all the left page,
Alt - selects all the right pages. 

Installation

Download the file MultiPaste.zxp and double-click it. Follow the instructions of Adobe Extension Manager.

Uninstall

Run Adobe Extension Manager. Find the item "MultiPaste" in the list and click "Remove" button.


***

Скрипт для Adobe Indesign CS4+

Вставляет содержимое буфера обмена на каждую из указанных страниц текущего документа. Щелкните меню Edit->MultiPaste… Выберие в списке нужные страницы и нажмите кнопку ОК. Удерживайте CTRL (или Command на Mac) для выбора нескольких страниц. Повторный щелчок на элементе списка снимает выделение.


Удерживайте клавиши перед запуском скрипта для дополнительных функций:

Shift - выделяет все страницы,
Ctrl - выделяет все левые страницу,
Alt - выделяет все правые страницы.


Установка

Скачайте файл MultiPaste.zxp и дважды щелкните по нему. Следуйте инструкциям программы Adobe Extension Manager.


Удаление

Запустите Adobe Extension Manager. Найдите пункт "MultiPaste" в списке и нажмите кнопку "Удалить".
